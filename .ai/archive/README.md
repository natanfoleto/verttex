# Arquivo Geral de Documentos — VERTTEX

> **Categoria:** Documentos Gerais Descontinuados ou Obsoletos  
> **Localização:** `.ai/archive/`

## Objetivo da Categoria
Esta pasta armazena documentos gerais do projeto que deixaram de ser válidos ou foram substituídos por versões mais recentes, preservando a rastreabilidade sem poluir as categorias vigentes.

## Regras de Arquivamento
1. Um documento só é movido para esta pasta após confirmação de substituição ou obsolescência formal.
2. Todo documento arquivado deve conter uma nota explicativa indicando a data de arquivamento, motivo e o documento substituto (quando houver).

## Relação com `.ai/roadmaps/archived/`
- **`.ai/archive/`:** Arquiva documentos gerais do projeto (especificações, estudos descontinuados).
- **`.ai/roadmaps/archived/`:** Arquiva exclusivamente roadmaps de execução cancelados ou substituídos.
