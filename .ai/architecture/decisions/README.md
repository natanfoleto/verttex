# Decisões de Arquitetura (ADRs) — VERTTEX NF

> **Categoria:** Architecture Decision Records  
> **Localização:** `.ai/architecture/decisions/`

## Objetivo
Esta pasta armazena o registro histórico de Decisões de Arquitetura (ADRs - Architecture Decision Records) do projeto VERTTEX NF.

## Estrutura de um ADR
Cada decisão arquitetural significativa deve seguir o padrão:
`ADR-XXXX-NOME-DA-DECISAO.md`

Exemplo de formato:
```markdown
# ADR-XXXX: [Título da Decisão]

## Status
[Proposta | Aprovada | Substituída por ADR-YYYY | Rejeitada]

## Contexto
Por que esta decisão é necessária?

## Decisão
O que foi decidido?

## Consequências
Quais são os impactos positivos e negativos desta decisão?
```
